QUICK START
===========
1. cd /sessions/ecstatic-vigilant-ramanujan/mnt/work/epo-saas/frontend
2. npm install  (already done)
3. npm run dev
4. Open http://localhost:3000
5. Press Ctrl+D to open demo controls

PRODUCTION DEPLOYMENT
======================
1. npm run build
2. npm start

The built files will be in the .next/ directory.

BACKEND REQUIREMENTS
====================
Backend must provide these endpoints:
- GET    /api/epos              -> EPO[]
- GET    /api/stats             -> Stats
- GET    /api/communities       -> Community[]
- GET    /api/vendors           -> Vendor[]
- POST   /api/demo/simulate-email (subject, body)
- POST   /api/demo/seed
- POST   /api/demo/reset

API base URL: http://localhost:8000 (configured in src/lib/api.ts)
