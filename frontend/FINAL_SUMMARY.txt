================================================================================
                         API INTEGRATION LAYER
================================================================================

All API functions in /src/lib/api.ts with error handling and fallback data.

Functions Implemented:
1. getEPOs() -> EPO[]
   - Fetches list of all EPOs
   - Fallback: empty array
   - Used in: EPOs page, Dashboard

2. getStats() -> Stats
   - Fetches: totalEPOs, captureRate, totalValue, needsFollowUp, changes
   - Fallback: default stats with zeros
   - Used in: Dashboard

3. getCommunities() -> Community[]
   - Fetches: name, captureRate, volume
   - Fallback: demo community data
   - Used in: Analytics page

4. getVendors() -> Vendor[]
   - Fetches: name, volume, amount
   - Fallback: demo vendor data
   - Used in: Analytics page

5. simulateEmail(subject, body) -> parsed EPO
   - POST /api/demo/simulate-email
   - Used in: Demo controls

6. seedData() -> success status
   - POST /api/demo/seed
   - Used in: Demo controls

7. resetData() -> success status
   - POST /api/demo/reset
   - Used in: Demo controls

API Base URL: http://localhost:8000 (hardcoded, can be moved to env)

Type Definitions in api.ts:
- EPO: id, vendor, community, lot, description, amount, status, createdAt, age
- Stats: totalEPOs, captureRate, totalValue, needsFollowUp, changes
- Community: name, captureRate, volume
- Vendor: name, volume, amount
